# Real-Time-Text-Sentiment-Analysis-using-Web-App
Real-Time Text Sentiment Analysis Web App – A lightweight Streamlit-based tool using TextBlob (NLP/lexicon-based ML) to classify text as Positive, Negative, or Neutral. Computes polarity &amp; subjectivity scores, supports text cleaning, single-input and CSV batch analysis, with downloadable results for real-time opinion mining.
